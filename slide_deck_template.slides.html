Effects of user category on the bike usage¶

Investigation Overview¶

In this investigation, I wanted to look at the some categories of bike users that could be affict on total usage. The main focus was on the four variables : duration, user type, birth year, member gender.

Dataset Overview¶

This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. this is the website of the data: https://www.bikeshare.com/data/ Go bike system contain 16 columns and 183411 rows. I remved some columns that I did not used.
